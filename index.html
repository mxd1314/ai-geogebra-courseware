帮助文档

欢迎使用 AI-GeoGebra 课件生成器！

  1. 配置AI服务：在左侧选择AI服务商并输入API Key
  2. 输入需求：用自然语言描述您想要的课件，或选择模板
  3. 生成课件：点击"生成课件"按钮，等待AI生成
  4. 预览调整：在右侧预览课件，可通过参数面板微调
  5. 保存导出：保存到本地或导出为各种格式

几何图形
  • "画一个圆心在原点、半径为3的圆"
  • "画一个边长为5的正三角形"
  • "画两个相切的圆，半径分别为2和3"

  • "画指数函数y=2^x和对数函数y=log2(x)"

  • "画函数y=x^2及其导函数"
  • "画函数y=x^3在x=1处的切线"
  • "显示y=sin(x)在[0,π]区间与x轴围成的面积"

常见问题

如何获取API Key？

OpenAI: 访问 platform.openai.com 注册并获取API Key

通义千问: 访问 dashscope.aliyun.com 申请API Key

文心一言: 访问 yiyan.baidu.com 申请API Key

生成的课件可以编辑吗？

可以！生成后您可以直接在GeoGebra画布中进行编辑，也可以导出为.ggb文件后在GeoGebra桌面版中继续编辑。

课件保存在哪里？

课件保存在浏览器的IndexedDB中，清除浏览器数据会导致课件丢失。建议定期导出备份。
